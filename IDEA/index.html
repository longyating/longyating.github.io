解决IDEA中Spring Boot启动报错Application startup failed

报错内容：

解决方法：

原因：工程要求的JDK版本与环境所配置的JDK版本不同所致。

pom.xml文件中所要求的为 1.8

导入工程后，查看默认的JDK，更改为1.8

  1. 路径为 File -> Project Structure ->Platform Settings ->SDKs 点击”+”号，添加版本为1.8的JDK所在的根目录

  2. 修改Project SDK 和Module SDK 为1.8
